# Smart Interview Preparation Portal

## Project Overview
Smart Interview Preparation Portal is a Java-based desktop application developed for students to prepare for placement interviews through aptitude quizzes, technical questions, and coding challenges.

---

## Features
- User Login System
- Aptitude Quiz
- Technical Questions
- Coding Challenges
- Result and Score Tracking
- Simple User Interface

---

## Technologies Used
- Java
- Java Swing
- File Handling
- Object Oriented Programming

---

## Project Modules
1. Login Module
2. Quiz Module
3. Result Module
4. Admin Module

---

## Software Requirements
- JDK 8 or Above
- Any Java IDE
  - Eclipse
  - IntelliJ IDEA
  - NetBeans

---

## How to Run the Project
1. Open the project in any Java IDE
2. Compile all Java files
3. Run Main.java
4. Start using the application
